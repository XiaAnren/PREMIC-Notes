### 正式安装

使用`npm`进行安装：
```bash
npm install -g @anthropic-ai/claude-code
```

### 基础配置

主要参考资料：<https://www.bilibili.com/video/BV1KJ61BBEB1>

Claude Code的配置文件默认位于`C:\Users\[USER]\.claude.json`（Windows）`$HOME/.claude.json`（Linux）。

使用时在需要的文件夹内：
```bash
claude
```

#### 取消地区限制

在配置文件内添加：
```json
{
  "hasCompletedOnboarding": true
}
```

#### 指定默认模型

模型标价：<https://docs.z.ai/guides/overview/pricing>

考虑使用Z.ai的免费模型GLM-4.7-Flash。

在上述网页内注册账号并登录，直接申请API Keys后填入配置文件内：
```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "",
    "ANTHROPIC_BASE_URL": "https://open.bigmodel.cn/api/anthropic",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "GLM-4.7-Flash",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "GLM-4.7-Flash",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "GLM-4.7-Flash",
    "ANTHROPIC_MODEL": "GLM-4.7-Flash"
  }
}
```
其中`ANTHROPIC_AUTH_TOKEN`对应所申请的API Keys，再次调用`claude`默认就是免费的模型。
