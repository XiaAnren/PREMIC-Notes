## 20260207-使用`Py-Spy`分析`Python`进程

### 问题描述

有同学反馈自己的`Python`进程卡死，经排查发现原因出在反复使用`np.save`向有大量文件的文件夹保存小数组。

### 使用`Py-Spy`分析`Python`进程

安装`py-spy`（`defalults`上的版本不够新，无法分析`3.12`及以上的`Python`进程）：
```bash
conda install py-spy -c conda-forge
```

输出分析信息：
```bash
py-spy dump --nonblocking --locals --pid <pid>
```

生成火炬图：
```bash
py-spy record -o profile.svg --nonblocking --idle -d 10 --pid <pid>
```
