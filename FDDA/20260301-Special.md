## 20260301-special未被正确同化

### 问题描述

- `obs_g2`所记录的观测数量过少。
- `obs_g3`缺失。

### 问题归因

- 没有对应修改`GMODJOBS/$GSJOBID/namelists/namelist.thin.d0?`。
- 缺失`GMODJOBS/$GSJOBID/namelists/namelist.thin.d03`。

### 解决方案

参考`GMODJOBS/$GSJOBID/namelists/wps.nl.template`中的模式区域配置进行修改。

<details>
<summary>补充说明</summary>

- `WRF_P`默认也会进行同化，可能需要考虑修改`fddahome/cycle_code/CSH_ARCHIVE/Forecast/special/RT_L_wrf_rtfdda.csh`中的相关逻辑：
  ```sh
  foreach i ( $DOMS )
    set ii = 0
    foreach f  (`/bin/ls $CYCDIR/*_s`)
      @ ii ++
      if($ii < 10) then
        ln -f -s $f OBS_DOMAIN${i}0${ii}
      else
        ln -f -s $f OBS_DOMAIN${i}${ii}
      endif
    end
  end
  ```

</details>
