## 20260228-special内文件过多

### 问题描述

`zout`中`cat`所有观测文件的日志信息被异常中断。

对应`fddahome/cycle_code/CSH_ARCHIVE/Forecast/special/RT_G_rtfdda_wrfqc_simple.csh`的如下内容：
```sh
set echo
cat $DATADIR/special/* >> ${obs}
unset echo
```

### 问题归因

文件数量过多（8w+）。

### 解决方案

改用`find`+`xargs`的组合：
```sh
find ${DATADIR}/special/ -maxdepth 1 -type l -print | xargs cat >> ${obs}
```
